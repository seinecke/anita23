#### Tutorial notebook
- Go to https://colab.research.google.com
- Select `Github`
- Enter link https://github.com/seinecke/anita23/blob/main/naima/naima_models.ipynb

- Go to https://colab.research.google.com
- Click `New notebook`
- (Install naima with `!pip install naima`.)
